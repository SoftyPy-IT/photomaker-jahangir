# Photo Frame Maker

A professional photo framing web application built with Next.js, TypeScript, and
HTML5 Canvas.

## Features

- **Circular Frame Selection**: Choose from 5 beautiful frame designs (Classic
  Gold, Modern Silver, Vintage Wood, Elegant Black, Ornate Bronze)
- **Smart Image Cropping**: Automatically crops uploaded images into perfect
  circles
- **Zoom Control**: Adjustable zoom from 50% to 200% for perfect positioning
- **Real-time Preview**: Canvas updates instantly when frames or zoom changes
- **High-Quality Export**: Download your framed photos as PNG files
- **Responsive Design**: Works perfectly on desktop and mobile devices

## Technology Stack

- **Next.js 16.1.6** (App Router)
- **React 19** with TypeScript
- **Tailwind CSS 4** for styling
- **HTML5 Canvas** for image rendering
- **Modern Browser APIs** for file handling
